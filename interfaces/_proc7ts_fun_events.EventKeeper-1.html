Interface EventKeeper<TEvent>

An event supplier that keeps the last event sent.

The registered event receiver would receive the kept event immediately upon registration, and all upcoming events after that.

Type parameters

  • TEvent: any[]

    An event type. This is a list of event receiver parameter types.
