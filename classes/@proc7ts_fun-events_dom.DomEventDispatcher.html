  • on<E>(type: string): OnDomEvent<E>
    • Defined in fun-events/src/dom/dom-event-dispatcher.ts:50

    Returns a sender of DOM events of the given type.

    The returned DOM event sender calls an EventTarget.addEventListener() to register listeners. But, in contrast, it allows to register the same listener many times.

    The event supply returned upon event listener registration unregisters it with EventTarget.removeEventListener() once cut off.
