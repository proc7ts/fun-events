on

  • on<E>(type: string): OnDomEvent<E>
    • Defined in dom/dom-event-dispatcher.ts:48

    Returns a DOM event listener registrar for the given event type.

    The returned DOM event listener registrar calls an EventTarget.addEventListener() to register listeners. But, in contrast, it allows to register the same listener many times.

    The EventInterest returned upon event listener registration, unregisters the given event listener with EventTarget.removeEventListener() when its EventInterest.off method is called.

    Type parameters

    • E: Event

      DOM event type.

    Parameters

    • type: string

      DOM event type name.

    Returns OnDomEvent<E>

    OnDomEvent registrar of DOM event listeners of the given type.
