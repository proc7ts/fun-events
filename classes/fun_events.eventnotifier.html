Class EventNotifier<E>

Event notifier can be used to register event receivers and send events to them.

It does not implement an OnEvent interface though. Use an EventEmitter if you need one.

Manages a list of registered event receivers, and removes them from the list once their supplies are cut off.

Can be used as EventSender.

Type parameters

  • E: any[]

    An event type. This is a list of event receiver parameter types.
